These directory contains some utilities needed during
the installation process.

Currently used utilities:

The "sleep.bat" file makes several batch files wait for a number 
of seconds. It is taken from:
http://www.robvanderwoude.com/wait.html

The T4ePortPing utility is used to check whether we can successfully
to the daisy repository and wiki. It is taken from the Tools4ever
Command Line Utilities Bundle, which can be obtained via:
http://www.tools4ever.com/products/free/command/
